# Minijava Compiler
This is a compiler for the language Minijava that I implementeed as a part of DOug Lea's CSC 444 course in Fall 2018.

$ cat HelloOne.minijava 
class HelloOne {
    public static io void main(String[] a){
        System.out.println(1);
    }
}
